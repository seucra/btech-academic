# P vs NP

## Class P
- P is the set of decision problems that can be **solved in polynomial time**.
- Example: Checking if a number is prime using deterministic algorithms.

## Class NP
- NP is the set of decision problems for which a **given solution can be verified in polynomial time**.
- Example: Sudoku solution verification.
